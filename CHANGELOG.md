# Changelog

## v1.2.0 — Kan & Dora Update
- 新增“鸣牌提示”开关。
- 鸣牌提示默认开启，每局开始自动重置。
- 关闭鸣牌提示后，不再弹出吃碰杠询问。
- 修复鸣牌提示关闭时的信息泄露问题：对手不会看到“正在思考”或等待状态。
- 新增暗杠功能。
- 手牌中有四张相同牌时，会出现可选的“暗杠？”按钮。
- 暗杠不会强制执行，不影响正常弃牌。
- 暗杠副露显示改为两端盖牌、中间明牌。
- 新增加杠功能。
- 已碰三张牌且手中有第四张时，会出现可选的“加杠？”按钮。
- 加杠会将已有碰牌升级为杠。
- 新增宝牌指示牌区块。
- 开局显示第一张宝牌指示牌，其余指示牌盖住。
- 隐藏宝牌指示牌改为干净空白牌背。
- 新增死壁 / 岭上牌逻辑。
- 普通牌山不再消耗死壁 14 张。
- 暗杠后立即翻开新的宝牌指示牌。
- 大明杠 / 加杠后不会立刻翻宝牌。
- 大明杠 / 加杠在摸岭上牌并打出一张牌后，才翻开新的宝牌指示牌。
- 修复杠后宝牌重复翻开的问题。
- 补充杠、宝牌、鸣牌提示相关测试。
- 测试通过：48/48。

## v1.1.0 — Meld Selection & River Clarity
- 新增吃牌多选，不再自动替你吃。
- 多种吃法时，玩家自己选择组合。
- 吃牌预览改用真实牌图展示。
- 副露加入横牌显示，更像真实日麻。
- 吃、碰、杠都会标出被叫的那张牌。
- 牌河新增手出标记。
- 摸切不标记，牌河更干净。
- 东家第一打默认摸切，不额外标记。
- 优化猜牌阶段文案。
- 优化自摸试抽阶段文案。
- 等待方不再看到无关操作按钮。
- 修复吃牌选择区嵌套按钮导致的 hydration error。
