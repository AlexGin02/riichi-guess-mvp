# Changelog

## v1.1.0 — Meld Selection & River Clarity
- 新增吃牌多选，不再自动替你吃。
- 多种吃法时，玩家自己选择组合。
- 吃牌预览改用真实牌图展示。
- 副露加入横牌显示，更像真实日麻。
- 吃、碰、杠都会标出被叫的那张牌。
- 牌河新增手出标记。
- 摸切不标记，牌河更干净。
- 东家第一打默认摸切，不额外标记。
- 优化猜牌阶段文案。
- 优化自摸试抽阶段文案。
- 等待方不再看到无关操作按钮。
- 修复吃牌选择区嵌套按钮导致的 hydration error。

## v1.0.0 — First Stable Release
- 摸牌分离显示。
- 东南家每局轮换。
- 猜牌历史公开展示。
- 已猜过的牌自动禁用。
- 鸣牌等待隐藏操作按钮。
- 牌河与猜牌区布局优化。
- 牌面视觉细节优化。
- 远程双人玩法稳定可用。

## v0.9.0-beta.1 — First Playable Beta
- 支持远程双人房间。
- 支持东家/南家玩家身份。
- 支持 Supabase Realtime 同步。
- 支持基础出牌、猜听牌、自摸挑战流程。
- 支持 Vercel 部署。
